# Windows 10 Acrylic

GTK theme based on the appearance of Microsoft's latest design guidelines

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
